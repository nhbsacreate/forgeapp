# FORGE - Personal Growth Feed

Your personal mental gym for strategic thinking and mental sharpness.

## EASIEST Deploy to Vercel (3 minutes - No Coding Required!)

### Option 1: Static Site (Simplest - Recommended for Quick Start)

1. **Download** the `public/index.html` file
2. **Go to** https://vercel.com/new
3. **Sign up** with GitHub (free)
4. **Drag and drop** the HTML file into Vercel
5. **Done!** Get your live URL instantly

Your app will be live at: `your-name.vercel.app`

### Option 2: Full GitHub Deployment (If you want version control)

### Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Name it: `forge-growth-app`
3. Make it Public
4. Click "Create repository"

### Step 2: Upload Files
1. In your new repo, click "uploading an existing file"
2. Drag ALL files from this folder
3. Click "Commit changes"

### Step 3: Connect to Vercel
1. Go to https://vercel.com/new
2. Sign up with GitHub (if you haven't)
3. Click "Import" next to your `forge-growth-app` repo
4. Click "Deploy" (no configuration needed!)
5. Done! You'll get a URL like `forge-growth-app.vercel.app`

## Updating Your Deployed App

**If you used Option 1 (drag & drop):**
- Just drag the updated HTML file to Vercel again

**If you used Option 2 (GitHub):**
- Push changes to GitHub → Vercel auto-deploys

## Custom Domain (Optional)

Want `forge.yourdomain.com`?
1. In Vercel dashboard, click your project
2. Go to "Settings" → "Domains"  
3. Add your domain
4. Follow DNS instructions
